File System
--------------------------------------
create a folder
```bash
mkdir foldername
```

delete a empty folder
```bash
rd foldername
rmdir foldername
rmdir "folder name with whitespaces"
```

delete a non-empty folder
```bash
rmdir /s foldername
rmdir /s "folder name with whitespaces"
```

create a file
```bash
type nul > filename.txt
```

delete a file
```bash
del filename.txt
del *.txt
```
